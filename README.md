# Garage Booking Application

A modern web application for managing and booking garage spaces. This application allows users to view available garages and make reservations in real-time.

## Features

- View list of available garages
- Real-time availability status
- Booking system with date and time selection
- Responsive design for mobile and desktop
- Modern UI with Vue.js and Tailwind CSS

## Tech Stack

- Frontend: Vue.js 3 with Composition API
- Styling: Tailwind CSS
- Backend: Laravel
- API: RESTful endpoints
- Database: MySQL (planned)
